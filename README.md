## Future Interns – Cyber Security Internship

### Prepared By

### Task

Task 1 – Vulnerability Assessment Report for a Live Website

### Objective

To identify common web application vulnerabilities, analyze security risks, and provide recommendations to improve the security posture of the target website.

### Tools Used

* Nmap
* OWASP ZAP (Passive Scan)
* Browser Developer Tools
* Canva

### Findings

* Open Ports Identified
* Missing Security Headers
* Directory Browsing Issues
* Information Disclosure Risks
* Medium and Low Risk Vulnerabilities

### Recommendations

* Implement security headers.
* Disable unnecessary information disclosure.
* Enable secure communication mechanisms.
* Perform regular vulnerability assessments and monitoring.
